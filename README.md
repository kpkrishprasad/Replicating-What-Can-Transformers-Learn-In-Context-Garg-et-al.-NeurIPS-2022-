# Replicating-What-Can-Transformers-Learn-In-Context-Garg-et-al.-NeurIPS-2022-
Replication of "What Can Transformers Learn In-Context?" (Garg et al., NeurIPS 2022). Trains a 12-layer GPT-2 transformer on 20D linear regression to verify transformers can learn gradient descent in-context. Includes curriculum learning, comprehensive baseline comparisons, and evaluation framework.
